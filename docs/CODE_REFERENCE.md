# Code Reference System Documentation\n\n## Overview\n\nThe Code Reference System is a comprehensive, searchable database of medical coding manuals integrated directly into SimCoder Pro. It provides real-time access to ICD-10-CM, CPT, and HCPCS Level II codes with complete descriptions, guidelines, and clinical context.\n\n## ICD-10-CM Complete Reference\n\n### Code Information Includes:\n- Official description from current manual (2025)\n- Long description with clinical context\n- Chapter and section organization\n- Includes notes (what conditions are included)\n- Excludes1 notes (codes that cannot be used together)\n- Excludes2 notes (codes that can be used together)\n- Code first / Use additional code requirements\n- 7th character options and requirements\n- Official coding guidelines\n- Clinical examples and use cases\n- Common mistakes coders make\n- When this code should be selected\n- Related codes and cross-references\n\n### 21 ICD-10-CM Chapters with Complete Organization\n\n## CPT Complete Reference\n\n### Code Information Includes:\n- Official CPT descriptor from AMA (2025)\n- Long description with clinical context\n- Category and type classification\n- RVU values (Work, Practice Expense, Malpractice)\n- Status indicator and special rules\n- Applicable CPT modifiers\n- Bundling rules and NCCI edits\n- Common billing scenarios\n- Typical unit values\n- Coverage information\n- Real-world examples\n- Related codes\n\n## HCPCS Level II Complete Reference\n\n### Code Information Includes:\n- Official description (2025)\n- Category and subcategory\n- Type (Drug, Device, Supply, Service)\n- Unit of measure\n- Coverage information\n- Applicable modifiers\n- Bundling rules\n- Prior authorization needs\n- Quantity limits\n\n## Search Features\n\n- Exact and partial code search\n- Description search with NLP\n- Filter by category and type\n- Search history and favorites\n\n## Code Validation\n\n- Active/inactive status checking\n- 7th character requirements\n- Invalid combination detection\n- NCCI bundle identification\n- Coverage verification\n\n## Workspace Integration\n\n- Right-side reference panel\n- Quick lookup during coding\n- One-click code insertion\n- Bundling and coverage alerts\n- Modifier suggestions\n\n## Regular Updates\n\n- ICD-10: October 1\n- CPT: January 1\n- HCPCS: Quarterly\n- Annual bundling updates\n
